# Notes
A command line wrapper on top of your favorite text editor to quickly take notes.

## Install
```
$ bash notes.sh
```

## Usage
```
# add something ...
$ notes "Implement a better application for taking notes"

# view or edit whatever you've noted so far
$ notes .

# if really needed...
$ notes --help
```
